- **GPT 系列**: GPT-2 → GPT-3 → GPT-4，自回归语言模型
- **LLaMA 系列**: Meta 开源模型，RoPE 位置编码、SwiGLU 激活
- **ChatGLM / Qwen / DeepSeek**: 国产大模型生态
- **MoE（Mixture of Experts）**: 稀疏激活，提升参数效率

### 3.3 微调技术（Fine-tuning）
- **全量微调（Full Fine-tuning）**: 更新全部参数
- **LoRA / QLoRA**: 低秩适配，大幅降低显存需求
- **Adapter / Prefix Tuning**: 轻量级参数高效微调
- **指令微调（Instruction Tuning）**: 遵循指令能力训练
- **RLHF / DPO**: 基于人类反馈的强化学习对齐

### 3.4 推理优化
- **量化（Quantization）**: INT8 / INT4 / GPTQ / AWQ / PPQ
- **剪枝（Pruning）**: 结构化/非结构化剪枝
- **蒸馏（Distillation）**: 知识蒸馏到小模型
- **KV Cache**: 推理加速关键
- **FlashAttention**: 高效注意力计算
- **vLLM / TensorRT-LLM**: 推理引擎

### 3.5 RAG（检索增强生成）
- **文档分块（Chunking）** → **向量化（Embedding）** → **向量检索** → **上下文增强生成**
- **向量数据库**: Milvus / Chroma / FAISS / Pinecone
- **混合检索**: 关键词 + 语义检索融合
